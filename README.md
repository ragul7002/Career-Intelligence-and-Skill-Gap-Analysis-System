# Career Intelligence and Skill Gap Analysis System

An AI-powered skill accelerator and career intelligence platform designed to help students analyze resume compatibility, identify skill gaps, and generate customized roadmaps to bridge those gaps.

## Project Structure
- `backend/` - Python Flask application for parsing, extraction, and evaluation logic.
- `frontend/` - React application built with Tailwind CSS and Vite.
